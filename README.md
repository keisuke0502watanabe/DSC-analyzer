# DSC-analyzer

Static HTML tools for visualizing and analyzing DSC data.

## Pages

- `webapp/dsc_analyzer_single.html`: Analyzer for single-file DSC workflows.
- `webapp/dsc_analyzer_db.html`: Analyzer focused on database-style datasets and related views.

## Usage

Open the HTML file you want to use directly in your browser:

1. Navigate to the `webapp/` directory.
2. Open either `dsc_analyzer_single.html` or `dsc_analyzer_db.html`.
3. Load your DSC data from the page UI.

## Notes

- Use `dsc_analyzer_single.html` for single-run analysis.
- Use `dsc_analyzer_db.html` when working with database-oriented datasets.
